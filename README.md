# Student Manager Plugin

## Giới thiệu
Plugin quản lý sinh viên trong WordPress.

## Chức năng
- Tạo Custom Post Type "Sinh viên"
- Nhập MSSV, Lớp, Ngày sinh
- Hiển thị danh sách bằng shortcode

## Cách sử dụng
- Cài plugin vào WordPress
- Thêm sinh viên
- Dùng shortcode:

[danh_sach_sinh_vien]

## Cấu trúc
student-manager/
 ├── student-manager.php
 ├── includes/
 └── assets/
